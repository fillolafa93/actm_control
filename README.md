# ACTM CONTROL
Optimal control freeway systems


## Prerequisites

- **Python 3.7+** 
- [**Poetry**](https://python-poetry.org/) installed globally

## Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/musel25/actm_control.git
   cd actm_control
    ```
2. **Install poetry**
   ```bash
   poetry install
    ```
3. **Run project**
   ```bash
    poetry shell
    python src/actm.ipynb
    ```
